  • Quais as diferençase ntre while, do-while efor
  • While - início ( antes de entrar no bloco ) - pré teste 0+ vezes
  • Do-while - final ( após o bloco de código ) - pós teste 1+ vezes
  • For - ideal para cenários onde: a) conheço o número de repetições; b) possuo variáveis de controle/contadores.
